# 📅 TimeNest– Smart Calendar with Memos & Events

A modern, minimal, and interactive calendar web app that lets users manage events, track date ranges, and maintain monthly memos — all in a clean UI.

---

## ✨ Features

- 📆 **Interactive Calendar Grid**
  - Monthly view with smooth navigation
  - Select single dates or date ranges

- 📝 **Monthly Memos**
  - Add, view, and delete notes for each month
  - Stored locally for persistence

- 🎉 **Event Management**
  - Add custom events (birthdays, holidays, etc.)
  - Visual indicators for quick recognition

- 🌙 **Dark Mode Toggle**
  - Seamless switching between light and dark themes

- 💾 **Local Storage Support**
  - Events and memos persist even after refresh

- 🎨 **Modern UI & Animations**
  - Smooth transitions and responsive layout
  - Clean, minimal, and user-friendly design

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **Date Handling:** date-fns  
- **State Management:** React Hooks  
- **Storage:** Browser LocalStorage  
- **Storage:** Browser LocalStorage

## 🚀 Getting Started

### 1. Clone the repository
```bash

```
### 2. Install dependencies
```bash
npm install
```
### 3. Run the app
```bash
npm run dev
```
